# 属性

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| value | 组件取值 | `number` | `0` |
| count | 星星的个数 | `number` | `5` |
| disabled | 是否变为只读状态 | `boolean` | `false` |
| half | 是否支持选中半星 | `boolean` | `false` |
| clearable | 是否可以清空值 | `boolean` | `false` |
